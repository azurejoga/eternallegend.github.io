o que é a eternal legend?

a eternal legend, é um grupo de cegos que quer trazer a acessibilidade em jogos para pessoas com cegueira,
seja por meio de entrar em contato com desenvolvedores, fazendo vídeos para seu canal do youtube
nós acreditamos que, se manter contato constante e disponibilizar as ferramentas necessárias pode ser feita uma boa acessibilidade em jogos

quando o grupo surgiu?

o grupo surgiu em 2018 graças algumas pessoas que queriam ver alguma mudança em jogos, ja que eles não tinham tantos jogos acessíveis para jogar, então eles diceram vamos então, (fazer com que eles fiquem acessíveis)

eles estão ativos até hoje?

sim! eles estão!

e como fazer com que os jogos fiquem acessíveis?

simples e prático.

  1. gravando vídeos para seu canal no youtube
  2. no seu canal no youtube eles fazem guias para pessoas cegas jogarem jogos jogáveis, mostrando jogos que poderiam ser também, potencialmente acessíveis, fazendo guia de menus, guia para jogos de luta, instalação de mods para habilitar acessibilidade em jogos e muito mais!
  3. enviando e-mails para desenvolvedores, ou falando com eles no discord ; ou qualquer outra plataforma de comunicação.

tolk, dll / biblioteca para usar leitores de tela em apps ou jogos no windows

explicação

Introdução
Tolk é uma extensão de aplicativo (DLL) que permite que aplicativos do Windows produzam texto por meio de software de leitura de tela (tecnologia assistiva para cegos e deficientes visuais). É uma camada de abstração sobre as APIs específicas do fornecedor que detecta automaticamente o leitor de tela ativo, permitindo um código de cliente limpo e simples. A saída de fala e braille é suportada em ambientes de 32 bits e 64 bits. Veja Supported screen readerspara mais detalhes. Além dos leitores de tela, a API de fala da Microsoft (SAPI) também é suportada. O nome Tolk é uma palavra holandesa que significa intérprete.
